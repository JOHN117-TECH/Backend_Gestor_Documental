Welcome to My API Files

Here you will find the following endpoint of the service that you want to consume

  • GET /files

    This endpoint returns all the files available in the service.

    Click here to see the response

    GET /files/:fileName

    Returns only one file corresponding to the provided fileName.

    Click here to see the response

    POST /files/upload

    Create a file and receive a file type field in the body

    PATCH /files/:id

    Updates the file depending on the given Id, receives the following json in the body, all fields are optional.
